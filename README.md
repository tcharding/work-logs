Work Session Logs  
=================  
  
Log files for day to day work sessions.  
  
* Logs are named by month.  
* Each log contains key to contents.  
* Print report with `./report month.md`  
  
Typical Categories  
------------------  
ops = system operations  
dev = development  
misc = miscellaneous  
read = read text book  
  
_see individual log files for topics_
